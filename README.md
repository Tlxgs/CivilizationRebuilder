# 《文明重建者》

《文明重建者》是一款文明模拟放置游戏。从帐篷和伐木场开始，逐步发展工业、科技，最终走向星际殖民。包含大量科技建筑、贸易系统、晶体装备、多种重置机制和永恒升级。

**游戏链接**：https://tlxgs.github.io/CivilizationRebuilder/

---

## 贡献指南

欢迎任何形式的贡献！无论是修复 Bug、优化性能、添加新内容，还是完善文案，都请在提交前遵守以下要求：

1. **代码规范**：保持现有代码风格（模块化、清晰命名、适当注释），避免引入未使用的依赖。
2. **功能测试**：所有改动请先在本地运行测试，确保不破坏现有游戏逻辑（尤其是生产计算、贸易系统、存档机制）。
3. **分支管理**：从 `main` 分支创建新的功能分支，完成后提交 Pull Request，并简要描述改动内容。
4. **内容平衡**：若添加新建筑、科技或晶体效果，请参考现有数值体系，保证游戏长期可玩性。
5. **文案语言**：UI 文本及注释请使用**中文**，保持统一、简洁的游戏风格。

感谢你的参与，一起让《文明重建者》变得更好！
